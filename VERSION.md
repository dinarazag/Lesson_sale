# История версий

## Ver 1.0 (базовая)

- Школьные предметы и английский язык
- Единый список уровней для всех предметов

## Ver 1.0.1

- Добавлены категории: кулинария, вокал, кинематографика, ораторское искусство
- Для языков — уровни по шкале CEFR (A1–C1)
- Для остальных предметов — начинающий / средний / продвинутый

## Ver 1.0.2 (текущая)

**Приоритет A — творчество и языки:**
- Танцы, рисование, фотография, гитара, актёрское мастерство
- Испанский, китайский, немецкий (CEFR)

**Приоритет B — школа:**
- Биология, история, обществознание, литература, география

**Приоритет C — дополнительные направления:**
- SMM, блогинг, монтаж видео, шахматы, логопедия
